Les cours que nous proposons aux enfants

Les cours sont accueillis à l’Institut Culturel Roumain chaque deuxième dimanche à partir du 30 avril, de 14h00 à 15h00. Les dates sont 30 Avril, 14 Mai, 28 Mai, 11 Juin, 25 Juin.

Scratch

Scratch a été créé par MIT et est une excellente introduction à la logique et les éléments de conception de la programmation informatique.

HTML/CSS/Javascript

HTML, actuellement sur sa cinquième version, est utilisé pour définir la structure d'une page Web. Accompagné de CSS et JavaScript décrira l'apparence et les actions de cette page.

Mobile

Le développement d'applications mobiles est un terme utilisé pour désigner l'acte ou le processus par lequel le logiciel d'application est développé pour les appareils mobiles.

PHP

PHP est un langage de programation généraliste et Open Source, spécialement conçu pour le développement d'applications web. Il peut être intégré facilement au HTML/CSS/Javascript.

Python

Python est un langage de programmation puissant et facile qui est utilisé dans le cadre des principaux sites Web, comme YouTube.

Ruby

Un langage de programmation dynamique et open source mettant l'accent sur la simplicité et la productivité. Il a une syntaxe élégante qui est naturel à lire et facile à écrire.

Raspberry Pi

Le Raspberry Pi est un mini-ordinateur à carte unique, développé par la Fondation Raspberry Pi avec l'intention de promouvoir l'enseignement de l'informatique de base dans les écoles.

Arduino

Arduino est une plate-forme de prototypage électronique open-source basée sur hardware et software flexibles et faciles à utiliser. Il est destiné à créer des objets ou des environnements interactifs.
